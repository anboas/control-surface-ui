Faithfulness Audit: What The Framework Handles vs. What Is Missing

This page intentionally recreates the reference diagram with current framework primitives, then names the gaps still visible under stress.

Stress result
Handled

Dense nested architecture surfaces

Headers, source/outcome rails, nested layers, service cards, platform bands, search/focus, layer toggles, and export hooks compose cleanly with framework classes.

Improved

Overflow-safe dense cards

Long labels now have explicit wrapping, card min-width contracts, and hover states that do not push outside the storage pipeline or medallion row.

Improved

Architecture connector grammar

The diagram now has export-visible directional rails between sources, workflow, fabric, and outcomes, plus storage pipeline arrows that behave like component primitives instead of decoration.

Partial

Domain icon fidelity

The registry now includes DNA, molecule, and microscope glyphs. It still needs richer branded Azure service symbols and a larger scientific/clinical icon family.

Remaining

Diagram schema export/import

This is still hand-authored HTML. Production needs a JSON diagram schema that generates stages, lanes, connectors, legends, layer controls, detail metadata, and export names.

Remaining

Auto-layout and export fidelity

The layout is CSS-tuned. Production should add fit-to-page, auto-spacing, label collision avoidance, route recalculation from data, and exact DOM/SVG export.
