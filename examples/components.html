Integration

Where Tailwind is still ahead: raw utility breadth, config-driven theme generation, ecosystem adoption, deep variant combinatorics, and fast arbitrary layout prototyping. This framework should answer those gaps with token-aware utilities, recipes, release discipline, and domain-specific components rather than becoming a Tailwind clone.
