# Policy Intelligence Plain HTML Starter

Use this starter when the downstream Policy MVP needs a no-build shell that consumes package artifacts directly.

## Setup

```bash
npm init -y
npm install control-surface-ui
```

Open or serve `index.html`. If you are using a local tarball, replace the install command with `npm install ./control-surface-ui-0.1.0.tgz`.

## What It Includes

- Policy route navigation: Overview, Graph, Documents, Review, Sources, Search, Workspace, Diagrams, and Data Model.
- Topbar utility cluster with search, notifications, account, and theme buttons.
- Overview KPI, table, and detail-panel composition.
- Adapter placeholders for policy records, autocomplete, graph layout, documents, review, sources, search, workspace, diagrams, and data model.

Use `docs/agent-handoff.md` and `docs/route-component-map.md` before expanding any route.
