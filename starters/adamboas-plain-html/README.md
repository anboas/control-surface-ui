# AdamBoas.com Plain HTML Starter

Use this starter when the downstream public site needs a no-build shell for homepage, services, profile, resume, insights, and contact content.

## Setup

```bash
npm init -y
npm install control-surface-ui
```

Open or serve `index.html`. If you are using a local tarball, replace the install command with `npm install ./control-surface-ui-0.1.0.tgz`.

## What It Includes

- Public navigation and hero section.
- Services, profile/resume, insights, and contact placeholders.
- Public search/filter hooks for writing and reference-loop content.
- Theme controls using the packaged framework behavior layer.

Use `docs/agent-handoff.md` before expanding this into the production AdamBoas.com site.
